# juan-diego
1. Uso de Cookies

Las cookies son pequeños archivos que se almacenan en el navegador del usuario. Se utilizan para guardar información como preferencias, configuraciones o datos de acceso, permitiendo que el sitio web recuerde al usuario en futuras visitas.

2. Creación de Sesiones

Las sesiones son mecanismos que permiten identificar a un usuario mientras navega por una aplicación web. Se utilizan principalmente para mantener al usuario autenticado después de iniciar sesión.

3. Eliminación de Sesiones

La eliminación de sesiones ocurre cuando el usuario cierra sesión o cuando la sesión expira automáticamente. Esto elimina los datos temporales almacenados y protege la seguridad de la cuenta.
